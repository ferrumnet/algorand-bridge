1. Contract Setup 
2. Admin Operations such as AllowTarget, DisAllowTarget & SetFee
3. Owner Operations such as AddSigner, SetFeeDistributor, SetTaxApp 
4. User Operations such as AddLiquidity & Swap Tokens
